# Research Topic

1. Use `browser_navigate` to search for the topic
2. Visit 2-3 relevant pages, extract key information
3. Synthesize findings into a clear summary
4. Save key facts to memory using `save_memory`

This skill requires deep reasoning and multi-step browsing — keep on paid APIs.
